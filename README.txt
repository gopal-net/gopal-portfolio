How to publish on GitHub Pages:
1. Create a GitHub account if you do not have one.
2. Create a new public repository. Example: gopal-portfolio
3. Upload all files from this folder to the repository root.
4. In GitHub, open Settings -> Pages
5. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
6. Save.
7. Wait 1-2 minutes. Your live site will appear there.

Important:
- Replace profile-placeholder.jpg with your own photo.
- Replace project-1.jpg and project-2.jpg with your real screenshots.
- The resume download button expects the file name: GOPALAKRISHNAN_D_Resume.pdf
